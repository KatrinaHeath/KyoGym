# Assets - Recursos de la aplicación

## Logo del Gimnasio

Coloca tu logo en esta carpeta con el nombre **`logo.png`**

### Recomendaciones:
- **Formato**: PNG (con fondo transparente) o JPG
- **Tamaño recomendado**: 200x200 píxeles o mayor
- **Nombre del archivo**: `logo.png`

### Formatos soportados:
- PNG (recomendado para logos con transparencia)
- JPG/JPEG
- BMP
- GIF
- SVG

Si tu logo tiene otro nombre, puedes:
1. Renombrarlo a `logo.png`
2. O modificar el código en `main.py` líneas 53 y 128 para usar el nombre correcto
